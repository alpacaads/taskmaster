How to use:

  1. Click Upload on any scene to pick a new image.
  2. The card updates instantly and the new image is previewed in the game (stored locally in your browser).
  3. Click Download to save the file with the correct filename (e.g. intro.png).
  4. Drop the downloaded file into zombie-rpg/images/ and commit to make it permanent.

Overrides are kept in localStorage only — they do not modify any committed files.
