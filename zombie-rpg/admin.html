How to use:

  1. Click ⚙ Settings once, paste a GitHub token, Save.
  2. Click Upload on any scene to pick a new image — previews live in the game.
  3. Click 🚀 Commit on a card (or 🚀 Commit all in the header) to push directly to zombie-rpg/images/ on the selected branch.
  4. GitHub Pages rebuilds in ~30–60 s — the committed image then serves to everyone.

No token? Use ⬇ ZIP to download and commit manually. Everything lives locally until you commit.
