How to use:

  1. Click Upload on any scene to pick a new image.
  2. The card updates and the new image is previewed in the game (stored locally in IndexedDB).
  3. Click Download to save the file as <id>.png.
  4. Drop the file into zombie-rpg/images/ and commit to make it permanent.

Overrides live only in your browser — they never modify committed files.
